# cli/

This directory has been removed as part of the platform consolidation (PLT track).

All CLI source code now lives in `platform/cli/`.

- Entry point: `platform/cli/index.ts`
- Commands: `platform/cli/commands/`
- Orchestrate sub-commands: `platform/cli/commands/orchestrate/`

The compiled binary entry point is `dist/platform/cli/index.js` (see `package.json` `bin` field).
